# Android Signing Quick Start

This is a quick reference for setting up Android APK signing. See [ANDROID_SIGNING_SETUP.md](./ANDROID_SIGNING_SETUP.md) for the complete guide.

## One-Command Setup

```bash
keytool -genkey -v -keystore "src-tauri/gen/android/app/upload-keystore.jks" -keyalg RSA -keysize 2048 -validity 10000 -alias upload -storepass android -keypass android -dname "CN=Tauri App, OU=Development, O=MyCompany, L=City, S=State, C=US"
```

## Create key.properties

File: `src-tauri/gen/android/key.properties`

```properties
storePassword=android
keyPassword=android
keyAlias=upload
storeFile=upload-keystore.jks
```

## Build and Install

```bash
# Build
npm run tauri android build -- --release

# Install
adb install src-tauri/gen/android/app/build/outputs/apk/universal/release/app-universal-release.apk
```

## File Locations

```

├── key.properties                    ← Create this
```

## Verify Setup

```bash
# Check keystore
keytool -list -v -keystore src-tauri/gen/android/app/upload-keystore.jks -storepass android

# Check files exist
ls src-tauri/gen/android/key.properties
ls src-tauri/gen/android/app/upload-keystore.jks
```
